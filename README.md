# Fractal explorer

Fractal explorer written in c++.  
It uses SFML and ImGui and is running on GPU.  

## Requirements

- A C++ compiler.
- CMake
- Make

## Usage

```bash
build.bat          # on Windows
./build.sh         # on Unix
```

Then

```bash
cd build

explorer.exe      # on Windows
./explorer          # on Unix
```
